Installation Instructions:

Macintosh OS X:
Drag the Buddi.app file into /Applications.

Windows:
Copy Buddi.exe to a folder of your choice; C:\Program Files\Buddi is 
recommended.  If desired, make a shortcut to add into the Start menu.

Debian / Ubuntu:
Double click on the Buddi.deb file.  When prompted, install the package.
Alternatively, you can install from the command line with the command
'sudo dpkg -i Buddi-X.deb' (where X is the version of the file)

Generic Unix:
Copy 'buddi' and 'Buddi.jar' to /usr/bin (or some other location in
your path).  Modify the buddi script to reflect the location you chose.
